# Stock Price Prediction

## Problem Statement
The goal of this project is to predict the closing price of a stock using historical trading data.

---

## Dataset
The dataset includes the following features:
- Opening Price  
- High Price  
- Low Price  
- Trading Volume  
- Closing Price  

Historical data was used to train and evaluate the regression model.

---

## Methodology
1. Data preprocessing  
2. Feature selection  
3. Train-test split  
4. Regression model training  
5. Prediction and evaluation  
6. Visualization of actual versus predicted prices  

---

## Model Used
Linear Regression

This model was chosen for its simplicity, interpretability, and suitability for numerical prediction tasks.

---

## Evaluation
- Mean Absolute Error (MAE) was used to evaluate prediction performance  
- Graphical comparison of actual and predicted values was performed  

---

## Conclusion
The regression model successfully captured general trends in stock prices, demonstrating the application of regression techniques in financial forecasting.
