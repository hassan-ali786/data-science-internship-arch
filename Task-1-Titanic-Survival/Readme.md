# Titanic Survival Classification

## Problem Statement
The objective of this project is to predict whether a passenger survived the Titanic disaster based on demographic and travel-related features such as age, gender, passenger class, and fare.

---

## Dataset
The dataset contains passenger information including:
- Age
- Gender
- Passenger Class
- Fare
- Embarkation Port

Missing values and categorical features were handled during the preprocessing stage.

---

## Methodology
1. Data loading and inspection  
2. Handling missing values  
3. Encoding categorical variables  
4. Feature selection  
5. Model training using Logistic Regression  
6. Performance evaluation using classification metrics  

---

## Model Used
Logistic Regression

This model was selected due to its interpretability and effectiveness for binary classification problems.

## Results
The model achieved good predictive accuracy and showed that:
- Gender had a significant impact on survival probability  
- Passengers in higher classes had higher survival rates
